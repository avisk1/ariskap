Flawfull

Flawfull is a web browser created by using Chromium and Electron with the JavaScript runtime environment NodeJS. It was released on October 14th, 2020 and continues to get updates every so often. Flawfull boasts a working search system with multiple search engines, a history tracking system, a tab system, and much more. Learn more.

Auto Door

Auto Door is a small door that's supposed to open and close automatically for cats to go in and out during the day. It will be built using an Arduino fitted with a motor, as well as a pulley system to help in the lifting process. Although it is currently in the planning stage, it should start coming together soon.

ChargeTech - Website

ChargeTech is a fictional business website created for a school business project. Its primary product is ConstaCharge, which is supposed to increase charging efficiency and save energy. Learn more.

Discord Bot "Todd"

"Todd" was a Discord bot created to experiment with Discord.js that ended up sending the Astronomy Picture Of the Day every day in a Discord channel. It, or "he", uses a Nasa API to request the current APOD and then uses a CronJob to send a Discord embed featuring the image, as well as its description, photographer, and date. He is also able to censor swear words, return the weather forecast in any city, and make use of a number of different APIs (such as a Pokemon database and a random advice-giver). Learn more.

Image Diluter
