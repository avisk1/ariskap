Discord Bot "Todd"
"Todd" was a Discord bot created to experiment with Discord.js that ended up sending the Astronomy Picture Of the Day every day in a Discord channel. He has a variety of different functions and commands, from telling the weather to Discord moderation.
APOD
Todd uses a Nasa API to request the current APOD and then uses a CronJob to send a Discord embed featuring the image, as well as its description, photographer, and date.
Weather
Todd can also request the current weather in any city by using a weather API. This includes displaying the temperature and the status of the clouds.
Discord Moderation
With Todd's moderation, any and all swear words will be censored, and roles can be added and taken away. Users can also be muted, kicked or banned by Admins all with a simple command.
Other APIs
Todd also boasts a collection of other interesting APIs, such as a Pokemon database, an advice-giver, and an API that checks and returns the astronauts currently on the ISS.
